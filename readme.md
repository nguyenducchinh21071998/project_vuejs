# Code base Laravel + VueJS (with VueCLI)
## Folder structure
```
.
├── app
│   ├── Console
│   ├── Exception
│   ├── Http
│   ├── Providers
├── bootstraps
├── config
├── database
├── docker
├── frontend
│   ├── vuejs
│   └── vuejs-ts
├── public
├── routes
├── scripts
├── storage
├── test
├── vendor
├── .dockerignore
├── .editorconfig
├── .env
├── composer.json
└── docker-compose.yml
```
## Run local
```
docker-compose build
docker-compose up
```
